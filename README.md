# Passenger Counter App

 
## Overview
Vanilla JavaScript passenger counter app built with HTML + CSS + JS. Tracks station entries/exits with increment, decrement, and save to log functionality. Perfect pre-React project!

## Features
✅ Live counter display
✅ +1 / -1 buttons for passengers
✅ Save entries to history log
✅ Reset counter
✅ Responsive design
✅ Pure vanilla JS (no frameworks)

- HTML5     | Semantic structure
- CSS3      | Flexbox + hover effects
- JavaScript| DOM manipulation + events

## Key Features Implemented
✅ Entry (+1) button → count++
✅ Exit (-1) button → count--
✅ Save → history.push(count)
✅ Reset → count = 0
✅ Live DOM updates

## Learning Outcomes
- DOM manipulation (textContent, innerHTML)
- Event listeners (addEventListener)
- Array methods (push, map)
- Template literals (`${count}`)
- CSS Flexbox responsive layout
